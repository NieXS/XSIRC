XSIRC by NieXS

GTK+ IRC client for Linux and Windows

XSIRC is a GTK+-based IRC client for Linux and Windows. It is a RFC 2812-compliant client, and has several useful features like network management and a link scraper.

Dependencies

* GTK+ 2.10 * Libgee-0.5 For compilation only: * valac * Python

Install

Installation is simple with waf: ./waf configure ./waf build ./waf install It can also be uninstalled by running ./waf uninstall
